  1. KinoDynamic A*

    1. 启发代价计算
      启发代价的计算需要计算一元四次方程和一元三次方程。一元四次方程使用费拉里方法，函数名quartic。一元三次方程判别式大于零使用求根公式，小于零使用三角函数法，函数名cubic。

  2. 哈密顿函数和Optimal Control Boundary Value Problem(OBVP 最优控制边界问题)
    哈密顿函数（Hamiltonian function）是一个在动力学中非常重要的概念，它描述了一个系统的总能量，包括系统的动能和势能。
    最优控制问题（Optimal Control Problem）是指在一定的约束条件下，使一个系统的某种性能指标（如能量最小或者时间最短）达到最优。在最优控制问题中，通常将目标函数表示为哈密顿函数，这种表示方式被称为哈密顿表述（Hamiltonian formulation）。
    具体来说，哈密顿表述将最优控制问题的目标函数（例如能量）表示为一个函数，这个函数包含了系统的状态、控制输入以及约束条件等信息。同时，哈密顿表述也引入了一个新的变量——共轭变量，用于描述系统状态和目标函数之间的关系。这种描述方式使得最优控制问题可以转化为一个变分问题（Variational problem），从而可以应用变分法等数学工具来求解。
    OBVP是Optimal Control Boundary Value Problem的缩写，即最优控制边界值问题。这是一类最优控制问题，其中目标是找到一个控制策略，使得一个系统从一个初始状态到达一个给定的目标状态，并在控制输入和状态变量满足一定的边界条件的情况下最小化一个给定的性能指标。

  3. Kinematic：运动学约束，例如避障。
    Dynamic：动力学约束，例如加速度(acceleration)、力(force)

  4. State Lattice Planning 状态栅格规划
      假设机器人并不能只表示一个质点，而需要充分考虑机器人的动力学和运动学约束。因此我们需要一个可行的运动连接的图，这种图也被称为Lattice Graph。
      建立这种图有两种方法，可分为控制采样和状态采样。Reverse direction：在状态空间(state space)中进行采样，将状态空间进行离散，找一条从当前状态到某一状态的可行的运动连接。
      关于如何仅通过两点状态，来得到最优的输入量u 和时间T ,请见下一章的OBVP原理讲解。
